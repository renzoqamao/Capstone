Administración de Redes

Auditoria de instalación

En este apartado se describe la utilidad de los servicios implementados hasta el momento.

DNS

Para el servicio DNS se utilizó un servidor CentOS7 con los paquetes net-tools y bind para la implemetanción de este servicio. El servidor DNS se encuentra en cada sede y se encarga de traducir las ip a dominios. Los dominios son sede.org sedecentral e informatica

DHCP

Para el servicio DHCP se utilizó un servidor CentOS7 con los paquetes net-tools y dhcp para la implemetanción de este servicio. El servidor DHCP se encuentra en cada sede y se encarga de asignar las ip correspondientes a cada oficina.
